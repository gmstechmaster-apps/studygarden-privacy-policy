This is the expected GitHub Pages URL for this repo (org: gmstechmaster-apps,
repo: studygarden-privacy-policy). To make it live:
  1. Push this repo to GitHub.
  2. Go to the repo's Settings -> Pages.
  3. Under "Build and deployment", set Source = "Deploy from a branch".
  4. Branch = main, folder = / (root). Save.
  5. GitHub will publish index.html at the URL above within a minute or two.

Once confirmed live, use this exact URL in:
  - Play Console -> Store presence -> Privacy Policy
  - Play Console -> App content -> Data safety
